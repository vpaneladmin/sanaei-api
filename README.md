# Sanaei Modern API
Write based on Laravel & PHP. For modern "Sanaei" users.

## Install

1. Clone this repo or download latest release
2. Copy files to your Sanaei server
3. Run `sudo install.sh`
4. Enter **IP Address**
5. Check server in `ipaddress:8009` is alive!
